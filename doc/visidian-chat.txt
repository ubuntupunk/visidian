1. Introduction                                           *visidian-chat-intro*

Visidian Chat provides an AI-powered chat interface that understands your notes
and can help you manage, analyze, and explore your knowledge base.

==============================================================================
2. Commands                                             *visidian-chat-commands*

:VisidianChat                                                  *:VisidianChat*
    Open the chat window to interact with the AI assistant.

:VisidianChatIndex                                             *:VisidianChatIndex*
    Index the current note and its linked notes in the vector store.

:VisidianChatIndexVault                                        *:VisidianChatIndexVault*
    Index all notes in the vault in the vector store.

==============================================================================
3. Chat Interface                                  *visidian-chat-interface*

The chat interface can be accessed using:
>
    :VisidianChat
<

This opens a split window with:
- Chat history at the top
- Input prompt at the bottom
- Context preview on the right

See |visidian-vectorstore| for details on context retrieval.


==============================================================================
4. Providers                                           *visidian-chat-providers*

Supported Providers:~
- OpenAI (default)
- Anthropic
- Custom provider (requires implementation)

Configuration:~
>
    " Set API provider
    let g:visidian_chat_provider = 'openai'
    
    " Set API key
    let g:visidian_chat_api_key = 'your-api-key'
<

==============================================================================
5. Usage Examples                                    *visidian-chat-examples*

Start a chat session:~
>
    :VisidianChat
    > What notes do I have about project planning?
<

Save chat history:~
>
    :VisidianChatSave
    > Enter filename: project-planning-chat.md
<

Use with context toggle:~
>
    :VisidianChat
    :VisidianChatContext
    > Summarize my notes about this topic
<
